# The Jarvice — Структура бизнес-презентации

## Слайд 1: Титульный
**The Jarvice**
AI-ассистент корпоративных сводок с нулевой утечкой данных
v0.2.1 · Privacy-first · Local-first

---

## Слайд 2: Проблема
Руководитель тратит 1-2 часа каждое утро на:
- Чтение почты (Exchange)
- Просмотр чатов (Teams)
- Сортировка важного от шума

**Риски:**
- Корпоративные данные в облаке (ChatGPT, Copilot)
- ПДн сотрудников у третьих лиц
- Compliance-нарушения

---

## Слайд 3: Решение
**The Jarvice** — утренняя сводка за 30 секунд вместо 2 часов

Как работает:
1. Скрейпит Exchange + Teams
2. Обезличивает ПДн ([ИМЯ_1] вместо Иванов)
3. Генерирует сводку через локальную LLM
4. Доставляет в Telegram

**Ключевой принцип:** ПДн никогда не покидают машину пользователя

---

## Слайд 4: Архитектура безопасности
```
Exchange/Teams → [RED/ PII-зона] → Анонимизатор → [GREEN/ Чистые данные] → LLM → Telegram
                  chmod 700                            │
                  Только владелец                 mapping.json
                                                  chmod 600
```

**Защита на каждом уровне:**
- RED (оригиналы) — chmod 700, доступ только владельцу
- GREEN (обезличенные) — имена, email, телефоны заменены масками
- Keychain — пароли и токены не в конфигах
- Log sanitization — чувствительные данные маскируются в логах
- Prompt hardening — защита от injection через системный промпт

## Слайд 6: Демо — от установки до сводки
**5 минут:**
```bash
bash setup.sh
the-jarvice configure --quick    # email + пароль + bot token
the-jarvice run --once
```

**Пример сводки в Telegram:**

## Слайд 7: Для Enterprise
**Интеграция:**
- Microsoft Exchange (EWS) — почта + календарь
- Microsoft Teams (IC3) — чаты + встречи
- Telegram — доставка сводок

**Безопасность:**
- On-premise: данные не покидают корпоративную сеть
- PII pipeline: RED/GREEN с audit log
- Keychain/keyring: FIPS 140-2 (macOS Keychain)
- Headless установка: `configure --non-interactive`

**Compliance:**
- ПДн не передаются третьим лицам
- Audit log всех операций с персональными данными
- Удаление: `uninstall` с secure wipe PII
