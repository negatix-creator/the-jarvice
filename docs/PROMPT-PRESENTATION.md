# The Jarvice — Дизайн-система и презентация

## Дизайн-система

**Концепция:** Cyber Shield — тёмный корпоративный tech-стиль. Щит = защита, замок = приватность, cyan = технологии.

**Палитра:**
- Фон: `#0A1628` (deep navy)
- Поверхности: `#0F2240` (dark slate)
- Акцент: `#00D4FF` (electric cyan) — заголовки, иконки, ссылки
- Бейджи: `#F0C040` (gold) — версии, CTA, важные числа
- Текст: `#FFFFFF` основной, `#8899BB` подписи, `#4466AA` dim
- OK: `#22C55E`, Warn: `#F59E0B`, Err: `#EF4444`
- PII Red: `#EF444480`, PII Green: `#22C55E80`

**Типографика:** Inter / SF Pro, моно: SF Mono / JetBrains Mono. H1 48–64px bold white, H2 32–40px, body 18–20px, code 16–18px cyan mono.

**Иконки:** Line art (stroke 1.5–2px), НЕ filled. Cyan для активных, slate blue для неактивных. Источник: Phosphor Icons (MIT).

**Карточки:** `background: #0F2240; border: 1px solid #00D4FF30; border-radius: 12px; padding: 24px; box-shadow: 0 0 20px #00D4FF10;`

**Не делать:** градиентный текст, скругления >16px, >3 цветов на слайде, filled иконки, анимации >0.5s, текст <14px, >6 строк в блоке.
